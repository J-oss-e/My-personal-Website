This folder is a minimal Next.js template created from the original `demo-react.html` demo. It contains a JSX conversion of the demo and a `data.js` module.

Quick notes:
- To see images in the demo you should copy the `public/images` folder from the original demo into `next-template/public/images`.
- The demo data is at `next-template/data/data.js` and the page is `next-template/pages/demo-react.jsx`.

Run locally:

1. Open a terminal in `Portfolio/14-BasicReact/next-template`.
2. Run `npm install`.
3. Run `npm run dev` and open `http://localhost:3000/demo-react`.

If you want the demo exactly like the original, copy the `images` directory from `Portfolio/14-BasicReact/public/images` into `next-template/public/images`.
